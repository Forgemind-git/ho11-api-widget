# HO11 Sample 3 — GitHub Stats Card

## Your task

As an open-source maintainer you keep opening GitHub to check a repo's health. Build a stats card from the GitHub public API.

## API to use

- GitHub REST API (no key for public repos) — `https://api.github.com/repos/{owner}/{repo}`

All free, no API key needed.

## What to display

- Stars, forks, open issues
- Last commit date
- Top contributor
- Language breakdown

## How to build it

1. Open index.html in your browser
2. Read the TODO comments
3. Add the fetch() call to the API
4. Render the data in the display section
5. Push to GitHub Pages for a live link
